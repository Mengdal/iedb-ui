# 入门
5分钟内即可完成 IEDB 安装和运行。

## 先决条件
+ Docker 最低 4GB 内存，建议 8GB 以上
+ 嵌入式配置推荐 内存1GB 4核A55 64位处理器 主频2.0GHZ 

## 快速入门
+ docker安装

```bash
docker run -d \
  --name arc \
  -p 8000:8000 \
  -v iedb-data:/app/data \
  371562656/iedb:latest
```

**确认程序正在运行：**

```bash
curl http://localhost:8000/health
```

## 获取您的管理员Token
首次启动时会生成一个管理员Token。**请立即复制——未来无法再次生成获取！**

+ **Docker**
+ **原生（systemd）**

```bash
docker logs iedb 2>&1 | grep -i "admin"
```

你会看到：

```plain
======================================================================
  FIRST RUN - INITIAL ADMIN TOKEN GENERATED
======================================================================
  Initial admin API token: xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
======================================================================
```

## 写入数据
+ **MessagePack**
+ **行协议**

```python
data = {
    "m": "cpu",
    "columns": {
        "time": [int(datetime.now().timestamp() * 1000)],
        "host": ["server01"],
        "usage_idle": [95.0],
        "usage_user": [3.2]
    }
}

response = requests.post(
    "http://localhost:8000/api/v1/write/msgpack",
    headers={
        "Authorization": f"Bearer {token}",
        "Content-Type": "application/msgpack",
        "x-arc-database": "default"
    },
    data=msgpack.packb(data)
)
```

## 查询数据
+ CURL

```bash
curl -X POST http://localhost:8000/api/v1/query \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"sql": "SELECT * FROM default.cpu LIMIT 10", "format": "json"}'
```

## 故障排除
+ **Docker**

```bash
docker logs iedb
```

+ **本地**

```bash
sudo journalctl -u arc -n 50
sudo systemctl status arc
```

### 常见问题
**身份验证错误**：请确保`TOKEN`已设置并包含在请求头中。

**未返回数据**：数据可能尚未刷新。清除缓存后刷新：

```bash
curl -X POST http://localhost:8000/api/v1/write/line-protocol/flush \
  -H "Authorization: Bearer $TOKEN"
```
